# HTML 演示文稿技能 - 详细指南

## 两种使用模式

### 模式一：主题生成（完整代写）

用户只提供一个主题，技能自动生成完整的 PPT 内容。

**用户输入示例：**
```
帮我做一个 PPT，主题是"微服务架构实践"
```

**技能执行流程：**
1. 询问使用场景和目标受众
2. 询问视觉偏好（主题风格、动画效果）
3. 生成 PPT 大纲供用户确认
4. 用户确认后，主 agent 调用子 agent 协作生成 HTML

### 模式二：Markdown 转换（格式转换）

用户提供已编写好的 Markdown 草稿，技能转换为 HTML 演示文稿。

**Markdown 草稿格式：**
```markdown
# 演示文稿标题

---

## 第一页：封面

- 副标题
- 作者：张三
- 日期：2026-03-25

---

## 第二页：目录

1. 引言
2. 核心内容
3. 总结

---

## 第三页：引言

这里是正文内容...

### 小标题

- 要点 1
- 要点 2
```

**Agent 执行流程：**
1. 主 agent 读取 `docs/skills.md` 检查格式规范
2. 调用 `chunk_splitter.py` 拆分 Markdown
3. 为每个 chunk 调用子 agent 分析内容类型
4. 根据内容类型调用子 agent 渲染（表格/图表/文本）
5. 主 agent 组装最终 HTML

---

## 主题风格详解

### minimalist（极简主义）
- 干净简洁，黑白灰 + 单强调色
- 适合技术分享、内部培训

### modern-gradient（现代渐变）
- 渐变色背景，阴影效果
- 适合产品发布、创意展示

### academic（学术专业）
- 稳重大气，衬线字体
- 适合学术汇报、正式场合

### creative（活泼创意）
- 色彩丰富，动效多
- 适合儿童教育、创意产业

## 动画效果详解

| 动画名称 | 效果描述 | CSS 实现 |
|---------|---------|----------|
| fade（淡入淡出） | 简单优雅，opacity 过渡 | `opacity: 0→1` |
| slide（滑动切换） | 类似传统 PPT，左右滑动 | `transform: translateX()` |
| zoom（缩放过渡） | zoom 效果，现代感 | `transform: scale()` |
| flip（3D 翻转） | 立体效果，视觉冲击强 | `transform: rotateY()` |

## 输出结构

单文件 HTML 输出：
```
presentation.html  # 所有内容内联（HTML+CSS+JS）
```

生成的 HTML 文件可直接在浏览器中打开，无需服务器。

## 幻灯片类型

| 类型 | 用途 |
|------|------|
| cover | 封面页（标题、作者、日期） |
| toc | 目录页 |
| section | 章节分隔页 |
| content | 内容页（文本、列表、图表） |
| code | 代码页 |
| image | 图片页 |
| table | 表格页 |
| end | 结束页（致谢、Q&A） |

|------|------|

| ← / ↑ | 上一页 |

## 注意事项

1. 生成的 HTML 可直接在浏览器中打开
2. 保持代码简洁易懂，方便用户后续修改
3. 主 agent 负责流程控制，子 agent 负责内容分析和渲染
4. 脚本仅作为底层工具使用
