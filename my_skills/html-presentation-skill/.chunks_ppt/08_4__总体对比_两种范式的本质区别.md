### 4. 总体对比：两种范式的本质区别

| 维度        | sci-group-read（Claude Code） | OpenClaw                    |
| --------- | --------------------------- | --------------------------- |
| **执行方式**  | 脚本驱动（Pipeline）              | 服务驱动（Service Orchestration） |
| **控制粒度**  | 极细（每一步可控）                   | 较粗（平台封装）                    |
| **透明度**   | 高（可调试）                      | 较低（黑盒）                      |
| **扩展方式**  | Skills + MCP                | 厂商生态                        |
| **自动化方式** | 手动编排自动化                     | 默认自动化                       |
| **适合人群**  | 极客 / 科研人员                   | 企业团队 / 产品研发                 |
